### 2.2 满意现秩序者（第6派系）

- 存储在 `data[106]`，开局为 **0**
- 不是真正的派系（没有 leader、ally 等属性），只有席位数
- 在饼图（Crushko.cs）中作为第6个扇区显示
- 总席位 = party_number[0..4] + data[106]
- 学说变动时增长：其他派系失去的席位一部分流向满意现秩序者

### 2.4 派系与执政

- `data[56]` = 当前执政派系索引，开局为 **1**（保守派）
- 单党制（data[15]<=7）：由 data[56] 直接确定执政
- 多党制（data[15]>=8）：比较盟友派系席位之和 vs 总席位，超过66%即为执政

### 2.5 派系相关计算

**饼图总席位**（Crushko.cs）：
```
total = party_number[0] + party_number[1] + party_number[2] + party_number[3] + party_number[4] + data[106]
```

**学说变更时的席位流转**（Doctrine_button_script.cs）：
1. 解除所有非保守派的盟友关系
2. 每个已启用且有席位的非保守派派系：交出 1/2 + 1/4 席位
3. 流失的席位归入保守派（party_number[1]）
4. 未启用的派系被启用
5. data[106]（满意现秩序者）累加部分流失席位

## 四、经济系统

### 4.1 预算机制 (Plusmisnus_script.cs)

**普通预算调整**（非贷款、非储备）：
- 点+：`data[类别] += 10`, `data[8] -= 10`
- 点-：`data[类别] -= 10`, `data[8] += 10`
- Shift: ±50, Ctrl: ±100
- 预算上限 planka2 = `(data[8] + data[36] + sum(data[71..81])) / 6`
- 经济学说 data[16] > 12 时上限进一步降低

### 4.2 贷款系统 (gosdolg)

**借贷**（点+）：
- `data[贷款索引] += 10`, `data[8] += 10`（增加预算）
- `data[1] -= 10`（党内支持下降）
- `data[4] += 25`（社会紧张度上升）

**还贷**（点-）：
- `data[贷款索引] -= 10`
- `data[8] -= 10/20/30`（按难度系数还钱）
- `data[1] += 10`（党内支持恢复）

**贷款上限** = `(empires[0].relations + empires[1].relations) / 5`

**贷款利息** = `data[69] / 40` 每回合，1980年后+1，1983年后+2

**贷款前置条件**：
- 加入经济联盟（ECON或经互会）或没有"落后经济"修正(modifies[12])
- 且拥有科技[9]("改进流水线") 或加入经互会 或加入东盟

### 4.3 外汇储备 (reserv)

- 存入：`data[储备索引] += 10`, `data[8] -= 10`
- 取出：`data[储备索引] -= 10`, `data[8] += 10`, `data[1] -= 10`, `data[3] -= 10`
- 储备影响：工业、服务业、生活水平，也会导致腐败增长

### 4.4 腐败影响

- 预算损失 = `data[26] / 10`
- 生活水平损失 = `data[26] / 50`

```

```

### 5.3 政治自由度计算 (hooray 公式)

```
hooray = (data[16]-9) + (data[15]-5) + (data[17]-15) + (data[50]-23) + (data[18]+data[51]-48)/2
```
附加：data[16]==11 或 10 时有额外奖励。

hooray 决定 data[14]（政体自由等级 0-5）。

## 六、军事系统

### 6.1 军队计算 (GetSoldiersNumber)

```
reservists = data[34] / 10
divisor = (data[51]==33) ? 3 : (data[51] - 29)
divisions = reservists / 2 + reservists / divisor
```

| 学说值 | 名称 | divisor | 效果 |
|--------|------|---------|------|
| 30 | 全面军事化 | 1 | 最多师 |
| 31 | 建设军力 | 2 | 中等 |
| 32 | 防御军 | 3 | 较少 |
| 33 | 合同军 | 3 | 较少（上限3） |

### 6.2 军事学说前置条件

- 30（全面军事化）：单党制下毛派执政，或多党制下威权/强硬路线
- 31（建设军力）：毛派/保守/温和执政，或威权/民主路线

- 33（合同军）：改革/自由派执政，或强硬/温和/民主路线
